FAQ

-¿Who is Luna the Crypto Cat?
Luna is a super cute cat who loves crypto and decided to create it's own ERC20 token on the Ethereum Network.

- ¿What is Luna Token (LUNA)?
LUNA is an ERC20 Token deployed on Ropsten, one of the test networks of the Ethereum Protocol. It can only be purchased in round numbers and has a total supply of 1,000,000 tokens. A total of 750,000 tokens will be sold via this Initial Coin Offering (ICO), Luna the crypto cat will keep the rest of the tokens.

- ¿How can i buy Luna Tokens?
First make sure that you are connected to the Ropsten test network using the Metamask browser extension, then select how many tokens would you like to buy and click the "Buy Tokens" button, a Metamask window will appear to confirm your transaction.

- ¿Are you stuck in a "Loading..." loop?
This may happen if you are not connected to Metamask, also this can happen if the network has many pending transactions and it's taking too long to confirm you purchase transaction. In both cases, refresh page after checking if you are connected to Metamask on Ropsten test network and see your pending transactions.
